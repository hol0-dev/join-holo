Features

Faster

Hol0 is written using FastAPI, a fast Python framework, so it is relatively fast.

Extensible

Hol0 allows you to create/use your own extensions and extend APIs, etc.

Controllable

Hol0 allows users to build their own servers. This allows users to have complete control over their data and content. In particular, it allows users to protect their privacy by severing ties with large companies that seek to use their data to learn and sell AI.

Federated

Hol0 can communicate with software using ActivityPub such as Misskey and Mastodon.

Misskey Compatibility

Hol0 is primarily focused on compatibility with Misskey, so it supports displaying reactions from software such as Misskey.

Open Source

Hol0 is open source and can be freely used and modified by anyone under AGPLv3.
